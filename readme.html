🔮 月影决策屋

在星辰与烛火之间，让命运为你指引方向

📋 项目概述

月影决策屋是一个完全独立的前端应用，包含两大核心功能模块：🔮 塔罗占卜 和 🎡 命运转盘。通过 DeepSeek API 实现智能解读，提供沉浸式的神秘体验。

✨ 功能特性

🔮 塔罗占卜

  • 78张完整塔罗牌（22大+56小阿卡纳）
  • 8种牌阵（单牌/三牌/爱情十字/事业金字塔/二选一/凯尔特十字/四季/生命之树）
  • 7+1问题分类选择器
  • 智能牌阵推荐
  • 解读前细节追问
  • DeepSeek思考模式深度解读
  • 多轮追问功能
  • 翻牌3D动画

🎡 命运转盘

  • 10个预设转盘
  • Canvas加权渲染引擎
  • 分类筛选
  • 自定义转盘
  • AI一句话生成转盘
  • OCR截图识别（UI占位）
  • 地理位置美食（UI占位）
  • 电影选择器（UI占位）

🚀 快速开始

  1. 直接用浏览器打开 index.html 即可运行
  2. 点击右上角齿轮图标打开配置面板
  3. 填写 DeepSeek API Key
  4. 选择塔罗占卜或命运转盘开始使用

⚙️ 配置说明

配置项 说明 默认值
API 地址 DeepSeek API 端点 https://api.deepseek.com
API Key DeepSeek API 密钥 （需用户填写）
模型名称 使用的模型 deepseek-v4-flash
思考模式 是否启用深度思考 开启
思考强度 思考深度 high

🎯 LLM 调用策略

场景 调用次数 说明
塔罗完整占卜 1次 含思考模式，翻牌+解读一次完成
塔罗追问（每次） 1次 按需调用
转盘AI生成 1次 无思考模式，响应较快
预设/自定义转盘 0次 纯前端实现

🎨 视觉特性

  • 深色神秘主题（紫/金配色）
  • 星空背景动画（150颗闪烁星星）
  • 月亮光晕动画
  • 选牌金色描边光效
  • 翻牌3D动画（Y轴旋转180度）
  • Canvas转盘物理旋转
  • 响应式设计（支持移动端）
  • 支持 prefers-reduced-motion

🔧 技术栈

  • 前端：纯 HTML/CSS/JavaScript，无框架依赖
  • 样式：CSS3 变量、动画、Grid/Flexbox
  • 图形：Canvas 2D API
  • API：DeepSeek Chat Completions API（OpenAI 兼容格式）
  • 存储：localStorage
